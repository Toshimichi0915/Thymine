# giz5
giz5 fixes small bugs and improve your combat experience
This mod requires Minecraft 1.16+, fabric loader and fabric api

### Features
1. FastSneak
2. FullBright
3. ToggleSprint
4. NoHurtBobbing
5. LowFire
6. SoftSneak
